# CareerNavigator 用户指南

CareerNavigator 是一个基于 AI 的智能职业规划助手，利用 LangGraph 工作流和阿里云百炼 API，为用户提供个性化的职业发展建议和行动计划。

## 🎯 核心功能

- **智能职业分析**: 结合用户画像、行业趋势和职位要求，进行深度匹配分析。
- **简历智能解析**: 支持上传简历图片，通过 OCR 自动提取个人信息和技能。
- **个性化目标拆解**: 将长远职业目标拆解为可执行的中短期目标。
- **详细行动计划**: 生成包含学习、实践、人脉建设的周/月度日程安排。
- **人机协作模式**: 在关键节点支持用户反馈，智能调整规划方向。

## 🚀 快速开始

### 1. 环境准备

- **Python**: 3.11+
- **API 密钥**: 阿里云百炼 (DashScope) API Key

### 2. 安装步骤

1. **克隆项目**
   ```cmd
   git clone <repository-url>
   cd coach
   ```

2. **安装依赖**
   ```cmd
   pip install -r requirements.txt
   ```

3. **配置 API 密钥**
   编辑 `start_server.bat` 或在终端设置环境变量：
   ```cmd
   set DASHSCOPE_API_KEY=your_actual_api_key_here
   ```

### 3. 启动应用

运行启动脚本：
```cmd
start_server.bat
```
服务启动后，访问 [http://localhost:5050](http://localhost:5050) 即可使用。

## 📖 使用流程

1. **输入基本信息**: 在首页填写您的年龄、学历、工作经验、当前职位、目标行业及职业目标。
2. **上传简历 (可选)**: 点击上传按钮，上传您的简历图片，系统将自动填充相关字段。
3. **开始规划**: 点击“开始规划”按钮，系统将启动智能工作流。
4. **查看分析报告**: 系统首先会生成一份深度分析报告。
5. **提供反馈**: 您可以对报告表示满意或提出修改意见。
6. **获取行动计划**: 确认分析后，系统将为您生成详细的目标拆解和日程计划。

## 🛠️ 常见问题

- **Q: 为什么启动失败？**
  - A: 请检查 `DASHSCOPE_API_KEY` 是否正确设置，以及 Python 版本是否符合要求。
- **Q: 简历解析不准确怎么办？**
  - A: 请确保上传的图片清晰，且包含关键的个人信息和工作经历。
- **Q: 如何修改已生成的计划？**
  - A: 在反馈环节提出具体的修改要求，系统会根据您的反馈重新生成。
